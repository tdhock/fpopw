# Changes in version 1.2 (2025-05-13) by Toby Hocking

- Fpop() gains verbose_file argument, which can be used to write candidates considered at each iteration to a file (default "" does not write any file, as previous version).
- Fpop()$J.est is now total cost minus penalty times number of segments (previous version was number of segments plus one).
